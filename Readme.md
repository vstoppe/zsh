# Installation

Install the ZSH by your package manager of trust:

  ``sudo pacman -S zsh``

Clone my git repo:

`git clone https://github.com/vstoppe/zsh.git ~/.zsh`

or

`git clone git@github.com:vstoppe/zsh.git ~/.zsh`

Call

`~/.zsh/setup.sh`

This will install:

- oh-my-zsh
- Powerlevel9k theme
